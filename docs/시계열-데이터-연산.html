4.2 시계열 데이터 연산

시간의 연산은 일반적으로 오늘로부터 며칠후, 언제로부터 몇개월전 등과 같은 연산을 의미한다. lubridate 패키지에서는 시간의 연산을 위해 몇개의 함수를 제공하는데 전과 후를 구분하는데에는 수학기호 +와 -를 사용한다.

today <- today()
today + 100   # 오늘부터 100일 후
## [1] "2021-05-14"
today - months(2)  # 오늘부터 2개월 전  
## [1] "2020-12-03"
today - years(1)   #오늘부터 1년전
## [1] "2020-02-03"
as.Date('2021-01-01') - as.Date('1980-01-01')  # 1980년 1월 1일 이후 날짜 수
## Time difference of 14976 days
